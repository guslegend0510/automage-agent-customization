### block_p0153_b003

8.3.3不可替员工做重大承诺

### block_p0153_b004

Staff Agent 不得替员工做超出个人范围的承诺。

### block_p0153_b005

不可做的事项包括：

### block_p0153_b006

1. 不得承诺任务一定按时完成。

### block_p0153_b007

2. 不得代表员工接受新任务，除非员工确认。

### block_p0153_b008

3. 不得承诺客户、合作方或外部对象。

### block_p0153_b009

4. 不得替员工确认涉及责任划分的内容。

### block_p0153_b010

5. 不得擅自修改员工已经确认的日报。

### block_p0153_b011

6. 不得替员工承认错误、承担责任或做绩效相关表态。

### block_p0153_b012

7. 不得自行判断员工表现优劣并上报为正式结论。

### block_p0153_b013

Staff Agent 可以提出建议，但最终确认必须由员工完成。

### block_p0153_b014

示例：

### block_p0153_b015

不推荐：

### block_p0153_b016

Staff Agent：该任务员工明天一定可以完成。

### block_p0153_b017

推荐：

### block_p0153_b018

Staff Agent：员工计划明天继续推进该任务，预计产出为接口字段草案，仍需后端确认部分字段。

### block_p0153_b019

MVP 阶段必须避免让Staff Agent 变成“自动替员工背书”的工具。

### block_p0153_b020

8.3.4不可越级读取他人数据

### block_p0153_b021

Staff Agent 只能读取与当前员工本人相关的数据。

### block_p0153_b022

可读取范围包括：

### block_p0153_b023

1. 员工自己的日报。

### block_p0153_b024

2. 员工自己的任务。

### block_p0153_b025

3. 员工自己提交的产出物。

### block_p0153_b026

4. 员工自己上报的异常。

### block_p0153_b027

5. 上级下发给该员工的任务说明。

### block_p0153_b028

不可读取范围包括：

### block_p0153_b029

1. 其他员工日报。

### block_p0153_b030

2. 部门完整汇总。

### block_p0153_b031

3. Manager 决策记录。

### block_p0153_b032

4. 老板决策卡片。
