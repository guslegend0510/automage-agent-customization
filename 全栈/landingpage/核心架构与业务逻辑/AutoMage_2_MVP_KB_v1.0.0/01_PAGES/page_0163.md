8.7.2Manager 向Executive 上推

### block_p0163_b008

Manager Agent 在以下情况下应向Executive Agent 上推：

### block_p0163_b009

1. 部门内无法解决。

### block_p0163_b010

2. 涉及跨部门资源。

### block_p0163_b011

3. 涉及老板目标或公司级优先级。

### block_p0163_b012

4. 影响关键交付。

### block_p0163_b013

5. 涉及预算或外部合作。

### block_p0163_b014

6. 多个部门出现同类风险。

### block_p0163_b015

7. 需要老板选择方案。

### block_p0163_b016

8. Manager 权限无法确认。

### block_p0163_b017

上推内容必须尽量整理成决策项，至少包含：

### block_p0163_b018

1. 决策标题。

### block_p0163_b019

2. 背景说明。

### block_p0163_b020

3. 影响范围。

### block_p0163_b021

4. 候选方案。

### block_p0163_b022

5. 推荐方案。

### block_p0163_b023

6. 推荐理由。

### block_p0163_b024

7. 来源部门汇总ID。

### block_p0163_b025

8. 紧急程度。

### block_p0163_b026

Manager 上推质量会直接影响老板侧体验。上推内容越结构化，Executive Agent 生成决

### block_p0163_b027

策卡片越稳定。

### block_p0163_b030

类型说明

### block_p0163_b031
