### block_p0237_b008

11.9.4重新打开异常

### block_p0237_b009

异常关闭后，如果问题再次出现，应允许重新打开。

### block_p0237_b010

重新打开条件包括：

### block_p0237_b011

1. 同一问题再次阻塞任务。

### block_p0237_b012

2. 关闭依据被证明无效。

### block_p0237_b013

3. 老板或Manager 要求重新处理。

### block_p0237_b014

4. 相关任务完成质量不达标。

### block_p0237_b015

5. 新数据表明风险仍然存在。

### block_p0237_b016

重新打开时，不应覆盖原关闭记录，而应新增状态更新：

### block_p0237_b017

{

### block_p0237_b018

"status": "reopened",

### block_p0237_b019

"reopen_reason": "任务虽已提交，但Executive Agent 联调时发现字段仍无法支持多方案决策记录。",

### block_p0237_b020

"reopened_by": 20001,

### block_p0237_b021

"reopened_at": "2026-05-05T10:00:00+08:00"

### block_p0237_b022

}
