6.10.3跨部门事项处理

### block_p0124_b027

如果某个阻塞事项涉及其他部门，Manager Agent 不能直接读取对方部门明细。

### block_p0124_b028

正确方式是：

### block_p0124_b029

1. 在blocked_items 中标记跨部门依赖。

### block_p0124_b030

2. 生成跨部门协作请求。

### block_p0124_b031

3. 由后端或Executive Agent 判断是否授权。

### block_p0124_b032

4. 必要时上推老板或更高层确认。

### block_p0124_b033

5. 授权后再读取必要的摘要级数据。

### block_p0124_b034
