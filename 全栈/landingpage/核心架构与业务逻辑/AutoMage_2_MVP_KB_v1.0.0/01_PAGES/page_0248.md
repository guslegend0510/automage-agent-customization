12.3.1Staff 日报

### block_p0248_b003

Staff 日报是员工个人工作记录，必须有员工本人确认。

### block_p0248_b004

Staff Agent 可以帮助员工整理日报，但不能替员工确认日报。员工确认前，日报只能是

### block_p0248_b005

草稿；员工确认后，才进入正式Work Record，并可被Manager Agent 汇总。

### block_p0248_b030

12.3.2Manager 汇总

### block_p0248_b031

Manager 汇总是部门级判断结果，需要部门负责人确认或至少标记确认状态。

### block_p0248_b032

Manager Agent 可以自动读取Staff Schema 并生成部门汇总，但部门健康度、Top 风险、

### block_p0248_b033

待上推事项、部门调整建议等内容会影响老板判断，因此需要明确责任主体。
