### block_p0154_b008

8.4Manager Agent 的决策边界

### block_p0154_b009

Manager Agent 是部门级节点，负责处理本部门的数据汇总、风险判断、异常处理和部

### block_p0154_b010

门内任务协调。

### block_p0154_b011

它的定位不是“高级日报生成器”，而是部门运行状态的判断节点。它需要区分哪些问题

### block_p0154_b012

可以在部门内处理，哪些问题需要上推老板。

### block_p0154_b013

Manager Agent 的决策边界可以概括为：

### block_p0154_b014

读取本部门数据；

### block_p0154_b015

生成部门汇总；

### block_p0154_b016

处理部门内事项；

### block_p0154_b017

上推超权限事项；

### block_p0154_b018

不越权处理老板级决策。

### block_p0154_b019

8.4.1可读取本部门数据

### block_p0154_b020

Manager Agent 可以读取本部门范围内的数据。

### block_p0154_b021

可读取内容包括：

### block_p0154_b022

1. 本部门员工Staff Schema。

### block_p0154_b023

2. 本部门员工任务状态。

### block_p0154_b024

3. 本部门异常记录。

### block_p0154_b025

4. 本部门历史汇总。

### block_p0154_b026

5. 与本部门相关的老板决策和任务。

### block_p0154_b027

6. 本部门产出物和工作记录。

### block_p0154_b028

读取时必须满足以下条件：

### block_p0154_b029

org_id 一致

### block_p0154_b030

department_id 在权限范围内

### block_p0154_b031

数据状态有效

### block_p0154_b032

数据未删除

### block_p0154_b033

当前Manager Agent 已绑定该部门

### block_p0154_b034

Manager Agent 不应读取其他部门员工明细。如果涉及跨部门问题，应通过上推或授权

### block_p0154_b035

流程处理，而不是默认拥有全公司读取权限。
