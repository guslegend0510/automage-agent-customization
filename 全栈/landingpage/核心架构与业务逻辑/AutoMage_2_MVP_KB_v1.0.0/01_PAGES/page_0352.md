# Page 0352

## 原始文本块

### block_p0352_b001

2026 年5 月3 日

### block_p0352_b002

16.11.2403 无权限

### block_p0352_b003

403 表示用户已认证，但没有权限执行该操作。

### block_p0352_b004

常见原因：

### block_p0352_b005

1. Staff Agent 读取他人日报。

### block_p0352_b006

2. Manager Agent 读取其他部门数据。

### block_p0352_b007

3. 普通员工确认老板决策。

### block_p0352_b008

4. 未授权Manager 创建跨部门任务。

### block_p0352_b009

5. 非负责人关闭high 异常。

### block_p0352_b010

6. Agent 节点与请求部门不匹配。

### block_p0352_b011

处理方式：

### block_p0352_b012

1. 立即拒绝。

### block_p0352_b013

2. 不自动重试。

### block_p0352_b014

3. 不更换参数重试。

### block_p0352_b015

4. 写入审计日志。

### block_p0352_b016

5. 必要时通知管理员或上级。

### block_p0352_b017

6. 给出清晰提示。

### block_p0352_b018

提示示例：

### block_p0352_b019

当前账号没有权限读取该部门数据。你只能查看自己负责部门范围内的汇总和任务。

### block_p0352_b020

Agent 收到403 后，禁止尝试修改department_id、user_id 或org_id 再次请求。

### block_p0352_b021

16.11.3404 不存在或不可见

### block_p0352_b022

对于敏感资源，后端可以返回404，而不是403，避免暴露数据是否存在。

### block_p0352_b023

适用场景：

### block_p0352_b024

1. 查询其他组织数据。

### block_p0352_b025

2. 查询无权限部门明细。

### block_p0352_b026

3. 查询无权限员工日报。

### block_p0352_b027

4. 查询已删除或不可见对象。

### block_p0352_b028

例如：

### block_p0352_b029

Manager A 查询其他部门某员工日报

### block_p0352_b030

→返回404

### block_p0352_b031

这样可以避免系统泄露“该员工日报确实存在”。
