# 设计升级说明 v1.2（Scroll Visual Upgrade）

## 升级范围

本次升级在既有工程上继续迭代，**未重建项目、未推翻首屏大框架**，重点重做首屏以下 section 的版式与视觉节奏。

## 关键改动

1. 新增视觉过渡段 `FilmStripGallery`
   - 斜向胶片式多轨滚动（不同速度）；
   - Hover 暂停；
   - 边缘蒙版渐隐；
   - 作为首屏与正文之间的 montage 段落。

2. Workflow 改为 `WorkflowStory` sticky 叙事
   - 左侧 sticky 主轴；
   - 右侧逐步说明（角色/API/表/状态）；
   - 不再是普通流程表罗列。

3. 三层 Agent 改为 `AgentNetworkSection`
   - editorial 文案 + 网络图；
   - 卡片 hover 展开关键 API 与关键表；
   - 保留 Staff/Manager/Executive 原有语义。

4. 新增视觉资料库 `VisualArchive`
   - 使用本地资产做 masonry 式展示；
   - 展示 risk/decision/workflow/dashboard 的界面切片；
   - 用于增强画册感与页面呼吸感。

5. Footer 升级为 `PremiumFooter`
   - 深色企业级信息架构；
   - Product / Architecture / Delivery / Resources 四列；
   - 底部明确主链路状态与唯一风险。

## 保留与一致性

- 保留首屏主结构与核心定位；
- 保留老板侧 P0 控制台、决策、任务、风险、联调状态；
- 保留唯一风险披露：`manager_cross_dept` 当前未拒绝（非阻塞）；
- 保留 demo fallback 与 API fallback 机制。
